설명
	1.	@Mapper:
	•	Spring Boot에서 MyBatis Mapper 인터페이스를 사용하려면 반드시 붙여야 합니다.
	•	Spring이 해당 인터페이스를 인식하고 매퍼 XML과 연결합니다.

	2.	어노테이션 기반 SQL:
	•	@Insert, @Select, @Update, @Delete 어노테이션을 사용하여 SQL을 직접 인터페이스에 정의할 수 있습니다.
	•	더 복잡한 SQL은 XML 매퍼 파일에 작성할 수 있습니다.

	3.	메서드 매핑:
	•	메서드의 매개변수는 SQL에서 #{}를 사용하여 바인딩됩니다.
	•	반환 타입은 SQL 실행 결과에 따라 단일 객체 또는 리스트로 설정합니다.

주의 사항
	•	어노테이션 기반 SQL 대신 XML 매퍼 파일에서 SQL을 정의한 경우, 해당 SQL은 매퍼 인터페이스에서 메서드 이름으로 호출됩니다.
	•	MyBatis의 XML 매퍼와 어노테이션 기반 SQL은 혼합해서 사용할 수 있습니다.
