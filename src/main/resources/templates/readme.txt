사용자 목록 출력 (<table>)
	•	th:each: 사용자 데이터를 반복하여 테이블의 행으로 출력합니다.
	•	${users}: 컨트롤러에서 전달받은 사용자 리스트를 Thymeleaf 템플릿에서 사용.

새 사용자 추가 (<form>)
	•	사용자 이름, 이메일, 비밀번호를 입력받아 새 사용자를 추가하는 폼.
	•	th:action="@{/users/add}": /users/add 경로로 POST 요청을 보냄.

사용자 삭제 (<form>)
	•	각 사용자에 대해 삭제 버튼 생성.
	•	th:action="@{/users/delete/{id}(id=${user.id})}": /users/delete/{id} 경로로 POST 요청을 보내 해당 ID의 사용자를 삭제.


5. 기능 요약
	1.	/users 페이지에서 사용자 목록을 확인할 수 있습니다.
	2.	새 사용자를 추가할 수 있는 폼이 제공됩니다.
	3.	각 사용자에 대해 삭제 버튼이 동작합니다.
	4.	모든 작업 후 페이지는 /users로 리다이렉트됩니다.
